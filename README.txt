Проект: "Модель двухколёсного робота на электроприводах с двухконтурными ПИД регуляторами"
В проекте есть 3 папки:

docs - отчет + содержание домашнего задания
Models - папка с моделями:
	Мodel_motor - модель электропривода
	Мodel_Robot - модель двухколёсного робота включающего электроприводы
src - папка с исполяемыми файлами, среди которых:
	clear_trash.m - скрипт отчистки (запускается при закрытии проекта)
	Motor_parametrs.m - скрипт содержащий исходные параметры мотора и предварительные вычисления (запускается при открытии проекта)
	Print_parameters.m - Shortcut, который выводит расчитанные параметры двигателя
	test_basic.m - папка, в которой лежит тест
	unload_workspace.m - скрипт отчистки (запускается при закрытии проекта)
